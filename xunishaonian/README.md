《虚拟少年》的歌词小程序

运行平台：
- Processing 3.5.4

使用场合：
- 通过视频会议软件云K歌时分享屏幕用，具体来说，就是先用OBS VirtualCam分享Processing的屏幕（这样就可以达到比很多视频会议软件的屏幕共享功能更高的帧率），然后用手机外放伴奏的同时，对着麦克风唱给会议里的人听。

安装方式：
- 首先安装Processing（3.5.4是我用的版本），这样就可以双击打开`xnsn.pde`
- 然后把方正兰亭细黑字体`fzltxh.ttf`放到`C:\temp`下（当然也可以替换成别的字体）
- 再安装`PixelFlow`和`Sound`两个库

使用方式：
- 歌词、逐字时间戳、自动换行时的每一行的倒计时都写在了`lyrics.pde`里，是一句一句组织的
- `J`与`K`切换到下一句与上一句
- `B` `N` `M` 用来录制当前这一行每一个字亮起/暗掉的时间；`B`标记亮起的时间，`N`标记暗下的时间并亮起下一个字，`M`只暗掉当前的字
  - 现在还很简陋，所以空格也被当成了一个字
- `V`清除当前这一行的逐字时间戳
- `P`印出逐字时间戳，这样就可以将用以上工序设定好的逐字时间戳复制粘贴到程序中
- `G`如果当前行有自动换行的倒计时，就开始自动换行。就这首歌来说，只有第一句歌词开始时才有自动换行
- `空格`停止自动换行，主要是在程序因为一些原因产生延时字幕滞后时使用
- `D`打开或关闭信息栏

以及，本次的每句的时间长度是用Audacity手工测量的，所以有些误差
